# FrayrBooth

Professional photo booth software for events, weddings, and activations. Built for kiosk and touchscreen use.

## Download

| Platform | File |
|---|---|
| Windows | `FrayrBooth Setup x.x.x.exe` |
| macOS | `FrayrBooth-x.x.x.dmg` |

Go to the [Releases](../../releases) tab to download the latest version.

## Install

**Windows**
1. Download the `.exe` installer
2. Run it — no admin required (per-user install)
3. Launch FrayrBooth from the desktop shortcut

**macOS**
1. Download the `.dmg`
2. Open it and drag FrayrBooth to Applications
3. Launch from Applications

## Features

### Capture Modes
- **Photo** — single shot or burst sequence (up to 8 shots)
- **Video** — timed video recording
- **GIF** — animated GIF from burst frames
- **Slow-Mo / 360°** — high-speed capture with speed control
- **Boomerang** — looping forward/reverse clip

### Camera Support
- Webcam / built-in camera
- Phone as wireless camera (connect via QR code on local network)
- Adjustable resolution (480p → 4K), frame rate, image quality
- Mirror, grid overlay, FPS counter
- Green screen / background removal (AI-powered)
- Photo filters (Grayscale, Sepia, Vivid, and more)

### Sharing
- QR code (guests scan to download their photo/video)
- Email sharing
- SMS sharing
- Configurable QR link expiry

### Printing
- Print directly to any connected printer
- Custom print layout designer (4×6, 5×7, 6×8, 2×6 strip)
- Add photo slots, text, logos to layouts
- Apply filters before printing

### Gallery
- Browse all captured media in grid or list view
- Filter by mode, date range
- Preview photos and videos full-screen
- Export entire gallery as ZIP
- Delete individual items or bulk delete

### Operator Dashboard
- PIN-protected operator access
- Per-event configuration (name, modes, countdown, sharing, kiosk)
- Dark / light theme toggle
- Configurable storage paths for media and layouts
- Reset settings to defaults
- Storage usage monitor

### Kiosk Mode
- Fullscreen mode for unattended booth use
- Hides system taskbar
- Session timeout with auto-return to idle screen
- Customisable idle screen (background image, logo, call-to-action text)

### Auto-Updates
- Automatic update checks on launch
- Downloads and installs updates in the background

## System Requirements

**Windows**
- Windows 10 or later (64-bit)
- 4 GB RAM minimum, 8 GB recommended
- Webcam or compatible camera

**macOS**
- macOS 11 (Big Sur) or later
- Intel or Apple Silicon (universal build)
- Webcam or compatible camera
